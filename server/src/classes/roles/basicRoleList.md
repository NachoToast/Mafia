# Base Roles

_These roles are top priority for implementation._

###### Town

0.  [x] Base
    -   No special abilities.
1.  [ ] Doctor
    -   Select one person a night to heal.
    -   Healing someone prevents them from dying to a basic attack.
2.  [ ] Jailor
    -   Select one person a day to jail.
    -   Jailed targets cannot do their night action or be attacked.
    -   You can anonmyously talk to your jailed target.
    -   You can chose to execute your jailed target (up to 3 times)
    -   If you execute a town member, you will not be able to execute again.
3.  [ ] Escort
    -   Select one person a night to roleblock.
    -   Roleblocked players cannot perform their night action.

###### Mafia

1.  [ ] Mafioso
    -   Vote for one person a night to attack.
    -   Votes will be overidden by the godfather if present.
    -   If no godfather is present, you become the godfather.
2.  [ ] Godfather
    -   Select one person a night to attack.

######

1.  [ ] Jester
    -   Convince the town to lynch you.
    -   If lynched, you can haunt one of your abstaining/guilty voters, killing them.
